# phpunit-annotation-testcase

Use simple annotations to mock the world!

## Installing Dependencies

Use [Composer][composer] and run :

```bash
$> composer require --dev yoannrenard/phpunit-annotation-testcase
```

## Requirements

* [Composer][composer]
* PHP >=5.3

## Run tests

```bash
$> bin/phpunit
```
